# ✈️ High Cloud Airlines Analysis

## 📌 Project Overview
This project analyzes airline data using multiple tools (Excel, SQL, Power BI, Tableau) to gain comprehensive insights into airline operations, customer behavior, and performance metrics.

## 🎯 Objectives
- Analyze airline performance
- Understand customer trends
- Improve operational efficiency
- Create interactive dashboards

## 🛠️ Tools Used

### 📊 Excel
- Data cleaning and preprocessing
- Pivot tables and charts
- Basic analysis and summaries

### 🗄️ SQL
- Data extraction and querying
- Joins and aggregations
- Filtering and transformation

### 📈 Power BI
- Interactive dashboards
- KPI tracking
- Data modeling and visualization

### 📉 Tableau
- Advanced visualizations
- Storytelling dashboards
- Trend and pattern analysis

## 📂 Dataset
Includes airline-related data such as:
- Aircraft groups
- Aircraft Types
- Airlines
- Carrier Groups
- Flight types
- airline_maindata.sql
- MainData_final.xlsx, etc.,


## 📊 Key Analysis Performed
- Flight performance analysis
- Revenue trends
- Customer segmentation
- Delay and cancellation patterns


## 📈 Key Insights
- Peak travel seasons identified
- Delay patterns across routes
- Revenue-generating routes highlighted
- Customer preferences analyzed
  

## 📌 Outcome
Developed dashboards across multiple platforms to provide a complete view of airline performance and support decision-making.

## 🚀 Future Improvements
- Real-time data integration
- Predictive analytics for delays
- Machine learning models for demand forecasting
